O [matplotlib](https://matplotlib.org/stable/index.html) é uma biblioteca abrangente para a criação de visualizações *estáticas*, *animadas* e até *interativas* em **Python**.
{: .text-justify}

Destaca-se as seguintes funcionalidades:

- Criação:

    * Desenvolver gráficos de qualidade de publicação com apenas algumas linhas de código;

    * Utilizar figuras interativas (zoom, pan, update);

- Customização

    * Fornece controle total aos estilos, propriedades das fontes, dos eixos, entre outros;

    * Exportação em diversos formatos;

- Extensivo:

    * Personalizado com diversos pacotes de terceiros;

Esta biblioteca fornece um interface similar a interface de geração de gráficos disponível no MATLAB, com a vantagem de ser ***gratuita e open-source***.
{: .text-justify}


<h2><a style="color:black">Licença</a></h2>

O **matplotlib** utiliza apenas código [BSD](https://pt.wikipedia.org/wiki/Licen%C3%A7a_BSD), que impõe poucas restrições quanto ao seu uso. Para o que vamos utilizar neste curso, basta citar que foi utilizado a biblioteca **matplotlib** e podemos utilizar todas as suas funcionalidades sem preocupação com direitos autorais.
{: .text-justify}

Para acessar os termos originais, clique [neste aqui](https://matplotlib.org/stable/users/license.html).

<h2><a style="color:black">Citação</a></h2>

O artigo que deve ser utilizado como citação para o uso do **matplotlib** é:

[J. D. Hunter, "Matplotlib: A 2D Graphics Environment", Computing in Science & Engineering, vol. 9, no. 3, pp. 90-95, 2007](https://ieeexplore.ieee.org/document/4160265)
{: .text-justify}

sendo que cada versão tem um DOI específico. Neste curso é utilizada a versão 3.3.4, que tem este DOI: [10.5281/zenodo.4475376](https://doi.org/10.5281/zenodo.4475376).
{: .text-justify}

Para citar o **matplotlib** na sua publicação, basta incluir na seção de **Material e métodos** que os gráficos foram desenhados utilizando o **matplotlib**.
{: .text-justify}

Exemplo:

> The graphics were drawn using matplotlib 3.3.4 software (Hunter, 2007).

> Os gráficos foram desenhados usando o software matplotlib 3.3.4 (Hunter, 2007).

Mais detalhes em [citing matplotlib](https://matplotlib.org/stable/citing.html).
